
# Next Quote App

互動式報價系統（Next.js + Tailwind）。

## 開發
```bash
npm install
npm run dev
# open http://localhost:3000
```


## Sheets / Airtable
1) 複製 `.env.example` 為 `.env.local` 並填上你的金鑰與 ID。  
2) Google Sheets：先在雲端硬碟建立一個試算表，把 **GS_CLIENT_EMAIL** 加到試算表共用為「編輯者」。  
3) 啟動開發伺服器後，頁面右側的「匯出到 Google Sheets / Airtable」按鈕即可送資料。
